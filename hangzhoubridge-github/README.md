# 桥·水·城 - 杭州古桥多维可视化系统

杭州古桥多维可视化系统，以45座古桥为核心数据，通过时空维度、水系网络、地理分布、美学展示四个维度，实现古桥文化的数字化呈现。

## 项目简介

本系统包含四大核心模块：
- 🏠 **首页**：精选古桥展示
- 🗺️ **古桥地图**：交互式地图（双模式）
- ⏳ **时光之旅**：时间线可视化
- 🏛️ **桥梁博物馆**：所有桥梁展示

## 技术栈

- **框架**：Next.js 15 + React 18 + TypeScript
- **可视化**：ECharts + Leaflet
- **动画**：Framer Motion
- **样式**：Tailwind CSS

## 本地运行

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
npm start
```
